# Design

## Summary

The site keeps the existing minimal, monochrome, text-first visual style.

## Visual System Guidance

- Preserve the current centered narrow page layout, white background, black body text, monospace typography, and ASCII art presentation.
- Preserve the existing spacing, responsive form layout, sticky inquiry section, and simple blue/gray button colors.
- CSS Modules should use semantic class names that describe component structure instead of utility-class names.
